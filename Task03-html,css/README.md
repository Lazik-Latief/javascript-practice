## HTML Basics – Page Structure Explained

1️⃣ <!DOCTYPE html>
Tells the browser that this document is written in HTML5.
It helps the browser render the page correctly.
Must always be the first line of an HTML file.

2️⃣ <html lang="en">
The <html> tag wraps the entire webpage.
lang="en" specifies that the page language is English.
Helps screen readers, search engines, and accessibility tools.

3️⃣ <head>
Contains meta information about the webpage.
Content inside <head> is not visible on the page.
Used to define:
Page title
Meta tags
CSS links
Fonts and icons

4️⃣ <body>
Holds all visible content of the webpage.
Everything users see (text, images, buttons, divs) is written here.
Only one <body> tag exists per page.

5️⃣ Meta Tags
<meta charset="UTF-8">
Supports all characters and symbols.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Makes the website responsive on mobile and tablets.
Meta tags help with SEO, performance, and compatibility.
