# JavaScript
---
## Objects & Functions Practice

1. This file covers JavaScript objects with different data types.
2. It shows how to create, update, merge, and access objects.
3. Nested objects and object freezing are practiced.
4. Object destructuring and renaming variables are included.
5. Symbols and Object.assign() usage is demonstrated.
6. Basic functions with parameters and return values are used.
7. Login and logout functions show real-life use cases.
8. Rest operator (...) is used to handle multiple arguments.
9. Functions receiving objects and arrays are practiced.
10. Overall focus is on core JavaScript fundamentals.
