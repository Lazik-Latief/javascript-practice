#  Color Generator Practice Project

This is a beginner-friendly practice project built using **HTML, CSS, and JavaScript**.  
It focuses on **DOM manipulation**, **loops**, **events**, and **basic styling**.

---

##  Features

- Dynamically generates a list of colors using JavaScript
- Each color item:
  - Has its own background color
  - Moves upward on hover (CSS animation)
  - Changes the page background color when clicked
- Clean and simple UI for practice

---

##  Concepts Practiced

### HTML
- Basic page structure
- Ordered lists
- Linking CSS and JavaScript files

### CSS
- Background & text colors
- Hover effects
- Transitions & transforms
- Box shadow
- Border radius

### JavaScript
- Arrays
- `for` loop
- DOM manipulation
- Event listeners (`click`)
- Dynamic styling
